## 🐛 Troubleshooting

### Extension not appearing?
- Check `chrome://extensions/`
- Make sure it's enabled
- Try reloading the extension

### "Invalid token" error?
- Generate new token from dashboard
- Make sure you copied the complete token
- Re-activate extension

### "Save to Genei" button not showing?
- Refresh the social media page
- Check extension is activated (click icon)
- Make sure you're on a supported platform

### Posts not appearing in dashboard?
- Wait a few seconds and refresh
- Check extension popup shows correct stats
- Try saving another post
