## Setup Instructions

### Prerequisites

1. Install Node.js (v18+): https://nodejs.org
2. Install Python (v3.8+)
3. Install Xcode Command Line Tools: `xcode-select --install`

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```
